## Main Project: arxgen

`arxgen` is a CLI project generator for creating Clean Architecture-style applications across multiple languages and frameworks.

Think of it as a starter machine for developers who want structure from day one.

```bash
npm install -g arxgen
```

Example:

```bash
arxgen create \
  --name student-api \
  --language typescript \
  --framework express \
  --entity student \
  --field name:string \
  --field email:string \
  --out ./generated
```

### What arxgen Can Generate

- Project starters
- Entities and CRUD modules
- DTOs, repositories, controllers, routes, and use cases
- Validation and pagination scaffolding
- Docker, Nginx, Redis, database, ORM, and JWT auth setup
- Extra modules for existing generated projects with `arxgen add entity`

### Supported Stacks

| Area | Technologies |
| --- | --- |
| Frontend | React, Vite |
| Node.js Backend | Express, NestJS |
| Backend Frameworks | FastAPI, Spring Boot, ASP.NET Core, Laravel, Gin, Rails, Ktor |
| Infrastructure | Docker, Nginx, Redis, GitHub Actions |
| Databases | PostgreSQL, MySQL, MongoDB |
