# 🚀 What is ArchGen?

ArchGen is a CLI-based architecture generator designed to create enterprise-grade backend structures using Clean Architecture principles.

Instead of wasting hours creating boilerplate code, folder structures, DTOs, repositories, and use cases manually — ArchGen automates the entire foundation.

Built for developers who care about:

- Scalability
- Maintainability
- Separation of Concerns
- Enterprise standards
- Clean code

# 🏗 Architecture

ArchGen follows strict Clean Architecture principles.

```txt
src/
├── domain/
│   ├── entities/
│   ├── repositories/
│   └── usecases/
│
├── application/
│   ├── services/
│   └── dto/
│
├── infrastructure/
│   ├── database/
│   ├── repositories/
│   └── external/
│
├── presentation/
│   ├── controllers/
│   ├── routes/
│   └── middlewares/
│
└── main/
```

# ⚡ Quick Start

## Installation

```bash
npm install -g archgen
```

---

## Create Project

```bash
archgen create my-app
```

---

## Run Project

```bash
cd my-app
npm install
npm run dev
```
